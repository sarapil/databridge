# DataBridge — Data Migration & Integration Hub
DataBridge handles CSV/Excel imports, cross-system migration (Odoo, QuickBooks, Sage, Tally, Salesforce), visual field mapping, and bidirectional data sync.

## Key Features
- **Import/Export**: CSV, Excel, JSON with visual field mapping
- **Migration Projects**: Step-by-step migration from legacy systems
- **Bidirectional Sync**: Real-time data sync with conflict resolution
- **Validation Rules**: Built-in and custom data validation
- **Rollback Points**: Safe data operations with undo capability
- **API Hub**: Connect to any REST API endpoint
