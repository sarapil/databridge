# Sync Configuration

Full sync configuration between source and target.

## How to use

1. Navigate to **DataBridge → Sync Configuration**.
2. Click **New** to create a record.
3. Fill in the required fields and **Save**.

## Common questions

**Q: Who can access Sync Configuration?**
A: Access is controlled by CAPS capabilities. Contact your administrator if you need access.

**Q: Can I export Sync Configuration data?**
A: Yes — use the List view **Export** button or the dedicated export report.
