# Getting Started

1. Go to **DB Settings** to configure import defaults and sync intervals
2. For imports: Create a **DB Import Job**, upload your file, map fields
3. For migration: Create a **DB Source Connection**, then a **DB Migration Project**
4. For sync: Create a **DB Sync Configuration** with field mapping
5. Monitor all operations in **DB Data Audit Log**
