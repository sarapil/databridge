# نقطة نهاية API

تعريفات نقاط نهاية API مع المصادقة والرؤوس.

## كيفية الاستخدام

1. انتقل إلى **ديتا بريدج ← نقطة نهاية API**.
2. اضغط **جديد** لإنشاء سجل.
3. أدخل الحقول المطلوبة ثم **احفظ**.

## أسئلة شائعة

**س: من يمكنه الوصول إلى نقطة نهاية API؟**
ج: يتحكم في الوصول محرك CAPS للصلاحيات. تواصل مع المشرف إذا احتجت وصولاً.

**س: هل يمكنني تصدير بيانات نقطة نهاية API؟**
ج: نعم — استخدم زر **تصدير** في عرض القائمة أو تقرير التصدير المخصص.
