# اتصال المصدر

بيانات اتصال قاعدة البيانات أو ملفات المصدر.

## كيفية الاستخدام

1. انتقل إلى **ديتا بريدج ← اتصال المصدر**.
2. اضغط **جديد** لإنشاء سجل.
3. أدخل الحقول المطلوبة ثم **احفظ**.

## أسئلة شائعة

**س: من يمكنه الوصول إلى اتصال المصدر؟**
ج: يتحكم في الوصول محرك CAPS للصلاحيات. تواصل مع المشرف إذا احتجت وصولاً.

**س: هل يمكنني تصدير بيانات اتصال المصدر؟**
ج: نعم — استخدم زر **تصدير** في عرض القائمة أو تقرير التصدير المخصص.
