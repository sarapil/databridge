# اتصال API

إعداد الاتصال بواجهات API الخارجية.

## كيفية الاستخدام

1. انتقل إلى **ديتا بريدج ← اتصال API**.
2. اضغط **جديد** لإنشاء سجل.
3. أدخل الحقول المطلوبة ثم **احفظ**.

## أسئلة شائعة

**س: من يمكنه الوصول إلى اتصال API؟**
ج: يتحكم في الوصول محرك CAPS للصلاحيات. تواصل مع المشرف إذا احتجت وصولاً.

**س: هل يمكنني تصدير بيانات اتصال API؟**
ج: نعم — استخدم زر **تصدير** في عرض القائمة أو تقرير التصدير المخصص.
