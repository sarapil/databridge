# مشروع الترحيل

مشروع ترحيل بيانات شامل يضم مهام متعددة.

## كيفية الاستخدام

1. انتقل إلى **ديتا بريدج ← مشروع الترحيل**.
2. اضغط **جديد** لإنشاء سجل.
3. أدخل الحقول المطلوبة ثم **احفظ**.

## أسئلة شائعة

**س: من يمكنه الوصول إلى مشروع الترحيل؟**
ج: يتحكم في الوصول محرك CAPS للصلاحيات. تواصل مع المشرف إذا احتجت وصولاً.

**س: هل يمكنني تصدير بيانات مشروع الترحيل؟**
ج: نعم — استخدم زر **تصدير** في عرض القائمة أو تقرير التصدير المخصص.
